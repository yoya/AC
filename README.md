# findall / res

script/findall.lua で item データを取得するのに、以下のようなシンボリックリンクが必要。

例 (WSL)
```
$ ln -s /mnt/c/Program\ Files\ \(x86\)/Windower\ Dev/res res
$ ln -s /mnt/c/Program\ Files\ \(x86\)/Windower\ Dev/addons/findAll  findAll
```
